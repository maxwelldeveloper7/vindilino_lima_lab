# **Atividade no laboratório de informática – Trabalhando com Arquivos**

## **1. Arquivo**

![](./assets/images/O_que_e_um_arquivo_.png)

Um **arquivo** é um lugar onde guardamos informações.

Essas informações podem ser:

* Textos
* Imagens
* Músicas
* Vídeos

📌 Pense assim:
Um arquivo no computador é como uma **folha ou documento**, mas em versão digital.

---

## **2. Antes dos computadores**

Antes de existirem computadores, as pessoas já guardavam informações, por exemplo:

* Em cadernos
* Em pastas
* Em gavetas com documentos

➡️ Isso também eram **arquivos**, mas físicos.

Hoje, usamos arquivos **digitais**, que ficam dentro do computador.

---

## **3. Tipos de arquivos**

Existem vários tipos de arquivos. Cada tipo serve para uma coisa.

Exemplos:

| Tipo de arquivo | Para que serve       |
| --------------- | -------------------- |
| Texto           | Escrever textos      |
| Imagem          | Ver fotos e desenhos |
| Áudio           | Ouvir músicas        |
| Vídeo           | Assistir vídeos      |

---

## **4. O que é extensão de arquivo?**

Todo arquivo possui um tipo, que é identificado pela **extensão**.

A extensão é o final do nome do arquivo.

### Exemplos:

* arquivo.txt → arquivo de texto
* foto.png → imagem
* musica.mp3 → áudio
* pagina.html → página da internet

📌 A extensão ajuda o computador a saber **como abrir o arquivo**.

---

## **5. Atividade realizada no laboratório**

Na aula, você realizou uma atividade prática no computador.

### **Passo a passo:**

1. Clicou com o botão direito do mouse na área de trabalho
2. Escolheu a opção para criar um arquivo de texto
3. Nomeou o arquivo como:

   ```
   arquivo
   ```
4. Abriu o arquivo usando o programa **Mousepad**
5. Digitou um texto
6. Salvou o arquivo
7. Fechou o programa
8. Abriu novamente o arquivo

---

## **6. O que você aprendeu com essa atividade**

Com essa prática, você aprendeu que:

✅ Um arquivo pode ser criado
✅ Podemos escrever dentro dele
✅ As informações ficam guardadas
✅ Podemos abrir e editar depois
✅ O computador precisa de um programa para abrir arquivos

---

## **7. Observação importante**

Mesmo que você não veja a extensão, o arquivo ainda tem um tipo.

Mais adiante, você vai aprender a usar extensões de forma mais avançada.

## **Conclusão**

Os arquivos são fundamentais para o uso do computador.
Tudo o que fazemos — textos, imagens, jogos — depende de arquivos.

Aprender a criar e usar arquivos é o primeiro passo para aprender programação.
